Privacy posture

Your photos stay where they belong.

Photo Cleaner processes your library locally. It only checks GitHub Releases when looking for app updates, and deletion always requires iOS system confirmation.

  • No photo uploads
  • Native iOS permission prompts
  • Open-source SwiftUI project
